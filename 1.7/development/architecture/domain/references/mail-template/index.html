GenerateThemeMailTemplatesCommand

PrestaShop\PrestaShop\Core\Domain\MailTemplate\Command\GenerateThemeMailTemplatesCommand Class GenerateThemeMailsCommand generates email themes templates for a specific language. If folders are not overridden in the command then MailTemplateGenerator will use the default output folders (in mails folder).
